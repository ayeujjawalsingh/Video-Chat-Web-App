Hello guys!!!
This is Video Calling Web App, where multiple people can join the
call & can do text chat with group members.
To RUN this project, your system must be needed Python3;
1 -- Install Python3 ( https://www.python.org/ftp/python/3.10.1/python-3.10.1-amd64.exe )
2 -- Install Django Framework,
        python -m pip install Django
3 -- Install Channels Modules,
        python -m pip install Channels
4 -- Run Code in Terminal of Parent Directory,
        python manage.py runserver
Then, click on Local Host Server link to open the project in browser

Congratulations!!! Your Project Have Been Run Successfully...
